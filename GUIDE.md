# DVC Systems Demo — Гайд по использованию

## Запуск

```bash
npm install
npm run dev
```

Откройте браузер: **http://localhost:5180**

---

## Основные разделы

### 🏠 **Панель директора** (по умолчанию)
Обзор состояния завода за 10 секунд:
- KPI (заказы, производительность, брак, загрузка)
- Граф производства (заказы на участках)
- Узкие места (красные участки требуют внимания)
- Ближайшие отгрузки

### 📊 **Dashboard**
- 8 ключевых показателей (KPI) с трендами
- График выполнения плана vs факт
- Круговая диаграмма статусов заказов
- Лента последних событий

### 🗺️ **Карта производства**
Участки завода в реальном времени:
- Зелёный = норма, жёлтый = требует внимания, красный = критично
- Нажмите на карточку участка → детали (работники, оборудование, очередь)

### 📋 **Канбан заказов**
Все заказы по этапам производства:
- **Перетаскивайте** карточки между колонками мышкой
- Нажмите на карточку → откроется полная информация
- Фото видны только при открытии заказа

### 📦 **Отслеживание деталей**
Поиск детали (например, "Б-17"):
- Откроется маршрут по всем этапам производства
- Показаны сроки, ответственные, комментарии

### ⚙️ **Контроль металла**
- Поступления и списания
- Складские остатки
- График расхода

### 👷 **Рабочее место мастера**
Для мастера участка "Покраска":
- Задачи на сегодня
- Кнопки: **Начать**, **Пауза**, **Завершить**, **Передать далее**
- **Фото** — прикрепить фотографию заказа
- **Проблема** — сообщить о проблеме (красный баннер на карточке)
- **Комментарий** — добавить комментарий в историю

## Карточка заказа

Откроется при клике на любой заказ. Вкладки:

| Вкладка | Что там |
|---------|---------|
| **Общее** | Номер, клиент, объект, статус, плановая дата |
| **Спецификация** | Марка металла, профиль, количество, вес |
| **Документы** | PDF, DXF (чертёж), XLS (металл) → нажмите **Открыть** или **Скачать** |
| **Фотографии** | Фото по видам (Готовность, Изделие, Контроль, Отгрузка) → клик = лайтбокс |
| **История** | Все действия по заказу + комментарии |

---

## Переключение ролей

Правый верхний угол → **Роль: Директор/Мастер/Технолог**

Каждая роль видит свой набор задач.

---

## Живой режим

**Кнопка "Живой режим"** (синяя, верхний правый угол):
- Вкл = система автоматически перемещает заказы между участками
- Вкл = KPI обновляются в реальном времени
- Видны в режиме реального времени — отлично для демо!

---

## Сброс данных

**Кнопка "Сброс"** в топбаре:
- Очищает все локальные изменения
- Восстанавливает исходные данные

Все изменения сохраняются в браузере (localStorage) и восстанавливаются при перезагрузке.

---

## Быстрые советы

✅ **Для впечатляющей демонстрации:**
1. Откройте **Панель директора** (первая страница)
2. Нажмите **Живой режим**
3. Смотрите, как KPI меняются в реальном времени
4. Кликните на красный участок в "Узких местах" → откроется **Карта производства**
5. Откройте любой заказ → вкладка **Фотографии**
6. Кликните на фото → откроется в полном размере

✅ **Тестирование функций:**
- Перетаскивайте карточки в **Канбане** → обновляется в реальном времени
- В **Рабочем месте мастера** нажимайте **Фото** и **Проблема** → вы сразу видите результат
- В карточке заказа → вкладка **Документы** → **Скачать** → скачается реальный PDF/Excel/DXF

## Что это на самом деле?

- **React 18** + **Vite**
- **Recharts** для графиков
- **jsPDF** для генерации PDF
- **Unsplash** для реальных фото
- **localStorage** как БД (все изменения сохраняются локально)
- Полностью работает в браузере, никакого сервера не нужно
