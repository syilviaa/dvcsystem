# DVC Systems Demo — ТОО «СТАЛЬ-ЭФФЕКТ»

Демонстрация системы управления производством металлоконструкций — «цифровой
двойник» завода, показывающий состояние производства в реальном времени.

## Запуск

```bash
npm install
npm run dev     # запуск на http://localhost:5180
```

Сборка для продакшена: `npm run build`, предпросмотр — `npm run preview`.

## Стек

- **React 18** + **Vite 5** — быстрый интерфейс, мгновенный HMR.
- **Recharts** — графики план/факт, статусы заказов, расход металла.
- Drag-and-drop канбана — нативный HTML5 (без внешних зависимостей).
- Дизайн-система на CSS-переменных (`src/styles/index.css`), светлая тема.

## Демонстрационные функции

- **Drag-and-drop** карточек в канбане с мгновенным обновлением статусов,
  KPI и ленты событий.
- **«Живой режим»** (кнопка в шапке) — имитация синхронной работы мастеров,
  технологов и директора: заказы автоматически движутся по этапам, обновляются
  KPI и уведомления в реальном времени.
- **Цветовая индикация участков** 🟢🟡🔴 по загрузке и задержкам.
- **Переключатель ролей** (Директор / Мастер / Технолог) в шапке.
- **Полная цифровая история заказа** — документы, фото, комментарии,
  ответственные, таймлайн действий.
- **Панель директора** — оценка состояния всего предприятия за 10 секунд.

## Данные

Все демо-данные (48 активных заказов, 7 участков, 36 сотрудников, склад металла,
детали, события) генерируются сид-генератором в `src/data/mockData.js` —
стабильны между перезагрузками.
